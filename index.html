PROJECTS

TOP

Kid Camp Guide

Technologies Used: Node.js, Express, Knex.js, PostgreSQL, Angular 4, Semantic UI

One place to find, compare and book Colorado youth camps. Parents usually have to spend a lot of time searching the web for camps for their kids. Now, they will have one resource to see the camps, create a list of favorite/ booked camps, and a calendar that adds your custom camp lists.

Battle Code Royale

Technologies Used: Node.js, Express, Knex.js, socket.io, PostgreSQL, Angular 4, Bootstrap

An interactive way to challenge a group of students/ peers with coding trivia questions. The admin will create a room, and then you can have multiple teams sign into the room and compete against each other. The admin will ask a question to the room and the teams will have 30 seconds to answer. The admin will score their answers which will be posted to the scoreboard. The admin can administer multiple rounds or finish the game.

Bee Nice

Technologies Used: JavaScript, jQuery, Node.js, Express, PostgreSQL, Materialze, Knex.js, HTML, CSS

An app for people who enjoy helping others. Create an account and start a list of ways to give back to others and the community. Here at the Bee Nice app, you can get some great ideas and even add your own. This app makes it easy to pay-it-forward and help make the world a better place.

Guide to the U.S. National Parks

Technologies Used: HTML, CSS, Google Maps API, Bootstrap, JavaScript, jQuery

A guide to the U.S. National Parks. I included a few maps to show all of the parks. I added markers that respond to a modal that provides a picture, title, established year, top features, and a link to each park's website. There is a select box to show a list of the parks and access the selected park's modal. There is also a way to create a favorites list.
